Scrapy ile Web Crawling

Scrapy, yapısal web içeriklerinin kolaylıkla crawl edilebilmesini sağlayan bir uygulama çatısıdır. Scrapy kullanarak yazacağınız çok kısa python betikleri ile dilediğiniz web içeriğini kolayca crawl edebilirsiniz.

Scrapy örümcekleri, yukarıda gördüğünüz gibi, scrapy.Spider sınıfndan türeyen python sınıflarından meydana gelir. Örümcek, komut satırı aracı tarafından çalıştırıldığında, sınıfın parse metodu çalışarak xpath veya css seçicileri tarafından yakalanan DOM elemanları, belirlenen biçime göre işlenir ve json çıktı üretir. Oluşturduğunuz örümceği komut satırı aracını kullanarak çalıştırmak için aşağıdaki komutu çalıştırın:

$ scrapy runspider myspider.py -o result.json

Komut çalıştırıldığında result.json dosyasına aşağıdaki içerik üretilir.

Scrapy İle Yeni Proje Oluşturma

Scrapy projeleri oluşturarak birden fazla örümcekten oluşan daha gelişmiş crawling yeteneklerine sahip projeler geliştirebilirsiniz. Yeni bir scrapy projesi oluşturmak için aşağıdaki komutu çalıştırabilirisniz:

$ scrapy startproject mysitecrawler

Komut çalıştırıldığında, dosya sisteminizde oluşacak dosya/klasör yapısı aşağıdaki gibi olacaktır.

mysitecrawler/
    scrapy.cfg            
    mysitecrawler/        
        __init__.py
        items.py     
        pipelines.py 
        settings.py  
        spiders/          
            __init__.py

Projenizin örümcek betikleri, spiders dizininde barındırılır. Dilerseniz aşağıdaki örnek örümcek betiğini kullanarak projemizin ilk örümceğini oluşturalım ve listing-spider.py olarak kayıt edelim.

Aşağıdaki komutu çalıştırarak listing isimli örümceğimizi çalıştıralım.

$ scrapy crawl listing

Scrapy Komut Satırı Arayüzü Kullanımı

Scrapy, basit örümcek sınıfları ve projeler dışında komut satırı arayüzüyle de kolaylıkla web içeriklerini işlemenize olanak sunar. Aşağıdaki komutu çalşıtırarak, ilgili urldeki içeriği komut satırı arayüzünden kolaylıkla işleyebilirsiniz.

$ scrapy shell 'https://www.example.com/satilik?viewType=List'
